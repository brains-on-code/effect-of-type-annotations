# Readme

`AllParticipants` includes the anonymized data of all participants for the study.

The `eye-tracking analysis tool` was used for the analysis of the eye-tracking data.

And the `Pilotstudie` folder contains the data collected during the pilot study.

`mainAnalysis.ipynb` contains the analysis run on the data and the plots used for the thesis.

`Eye Tracking Software` contains the program used for running the study on the lab PC. This was provided by the chair and slightly modified to better fit my study.
